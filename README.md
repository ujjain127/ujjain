# Ujjain Sriganesh - Minimalist Portfolio

A modern, dark-themed animated portfolio website inspired by aladesign.cz, built with React and Framer Motion.

- **Dark Minimalist Design**: Sleek black background with white text and refined typography
- **Interactive Hero Animation**: Full-screen hero with animated name reveal
- **Expandable Sections**: Click to expand/collapse research papers and content areas
- **Grid-Based Layouts**: Modern grid system for projects and skills showcase
- **Smooth Animations**: Sophisticated transitions powered by Framer Motion
- **Wavy Text Effects**: SVG-based text distortion for visual interest
- **Responsive Design**: Fully responsive layout optimized for all screen sizes

Inspired by contemporary portfolio designs, this site features:
- Minimalist aesthetics with maximum negative space
